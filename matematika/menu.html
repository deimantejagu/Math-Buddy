Veiksmai su skaičių aibėmis Skaičių aibių pavyzdys:
A = {3; 4; 5; 7}
B = {2; 4; 7} Sankirta:
A ∩ B = {4; 7} Sąjunga:
A ∪ B = {2; 3; 4; 5; 7} Skirtumas:
A \ B = {3; 5} Skirtumas:
B \ A = {2}
Aibių tipai ir jų pavyzdžiai N - natūralieji skaičiai (1; 2; 3) Z - sveikieji skaičiai (-1; -2; 0; 1; 2) Q - racionalieji skaičiai (Z aibė + trupmenos (a = m/n; m ∈ Z ir n ∈ N)) I - iracionalieji skaičiai (√3; Π - pi; e - Oilerio skaičius (a ≠ m/n; m ∈ Z ir n ∈ N) R - realieji skaičiai (Q + I)
Iracionalumo įrodymas Įrodyti: √2 iracionalus skaičius

1) Darome prielaidą, kad √2 - racionalus skaičius

2) Reiškinį √2 = m/n keliame kvadratu:
(√2 = m/n)^2

3) Reiškinį 2 = (m/n)^2 padauginame iš vardiklio:
(2 = (m/n)^2) * n^2

4) Gauname: 2n^2 = m^2

5) Iš šio reiškinio (2n^2 = m^2) matome, kad m dalijasi iš 2

6) m prilyginame 2k:
m = 2k

7) Todėl 2n^2 = (2k)^2

8) Gauname: 2n^2 = 4k^2

9) Reiškinį 2n^2 = 4k^2 padaliname iš 2
(2n^2 = 4k^2) / 2

10) Gauname: n^2 = 2k^2

11) Iš šio reiškinio (n^2 = 2k^2) matome, kad n taip pat dalijasi iš 2

Vadinasi, m/n yra suprastinama trupmena, nes m ir n dalijasi iš 2. Mūsų prielaida neteisinga

Išvada: √2 yra iracionalusis skaičius
